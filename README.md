## РЕШЕНИЕ ТЕСТОВГО ЗАДАНИЯ КОМПАНИИ YADRO
### Команда разработки подсистемы GSM-базовой станции


## Описание задания

Программа читает CSV-файл с таблицей, содержащей числа и формулы вида `=A1+B2`. Вычисляет значения всех формул и выводит результат в CSV-формате в консоль.


## Алгоритм

1. **Парсинг CSV** – построчное чтение файла, разбиение на ячейки по запятым

2. **Хранение данных** – трёхмерный массив `mass[строка][столбец][символ]`

3. **Подсчёт формул** – находим ячейки с формулами, увеличиваем счётчик `form_count`

4. **Решение формул** (многопроходный алгоритм):
   - Ищем ячейку с формулой
   - Смотрим ссылки в формуле (например `A1`, `Cell30`)
   - Если ссылка ведёт на другую формулу – пропускаем эту ячейку (будет решена позже)
   - Если по ссылкам доступны числа – решаем формулу
   - Заменяем формулу на вычисленное число
   - Уменьшаем счётчик `form_count` на 1
   - Повторяем, пока все формулы не будут решены

5. **Вывод** – CSV-формат с пустой левой верхней ячейкой


## Структура проекта
- main.c – точка входа, запуск тестов
- unit.c – основная логика (парсинг, вычисления)
- unit.h – заголовочный файл
- Makefile – сборка проекта
- example.csv – тестовый файл 1
- example2.csv – тестовый файл 2  



## Сборка и запуск

### Требования

- Компилятор GCC или MinGW
- Make (опционально)
## Установка и запуск
